# ⚙️ Instalação do Projeto
## Para a execução do projeto, é necessário ter o [Node.js](https://nodejs.org/pt) instalado

## 1️⃣ Clonar o repositório
```bash
git clone https://github.com/Time2-Procurai/procurai-frontend

```
## 2️⃣ Acessar a pasta do projeto
```bash
cd procurai-frontend

```

## 3️⃣ Instalar dependências
```bash
npm install

```

## ▶️ Rodando o Projeto
```bash
npm run dev

```
